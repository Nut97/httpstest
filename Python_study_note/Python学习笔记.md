#### 四、语法：

1. ##### 变量与常量：

   ```
   常量：具备字面的含义，可以按照字面上的含义使用它们的值，这就是常量。常量不能改变
   
   变量：为了方便调用这部分内存中存储的信息，为其设置一个名称，即变量名。
   ```

2. ##### 基本数据类型：

   ```
   整型(int):	 	n=10
   浮点型(float):		n=
   字符串(str):		data='abc'
   布尔值(bool):		b1=True
   ```
